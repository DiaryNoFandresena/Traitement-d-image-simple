DOCUMENTATION DES FONCTIONS OPENCV

Le traitement d'images avec C++ et OpenCV est un domaine passionnant et vaste. OpenCV (Open Source Computer Vision Library) est une bibliothèque open source largement utilisée pour le traitement d'images, la vision par ordinateur et la vision par machine.

cette programme permet de traiter et d'améliorer une image ou de connaitre plus sur l'image.

Configuration de la bibliothèque OpenCV C++

WINDOWS

Installation

  1. Téléchargez MinGW et OpenCV
  2. Installez MinGW dans C:\mingw et décompressez opencv-4.8.0.zip
  3. Ajoutez à la variable environnement Path: C:\mingw\mingw64\bin et C:\opencv\mybuild\bin
  4. Ajoutez les variables système: OPENCV_DIR-C:\opencv\mybuild et OPENCV_HOME-C:\opencv

Compilation

g++ -Wall -o "%e" "%F" -Ic:\opencv\include -Lc:\opencv\mybuild\lib -lopencv

LINUX

Installation

  1. sudo apt update
  2. sudo apt upgrade
  3. sudo apt install libopencv-dev

Compilation

g++ -o main nomDuFichier.cpp $(pkg-config --cflags opencv4) $(pkg-config --libs opencv4)

Lancement

./main

voici tout les fonctions utilisée dans cette programme pour vous familiariser

convertToGray

Description : Cette fonction convertit une image couleur en une image en niveaux de gris.

Utilisation : convertToGray(inputImage, outputImage);

Arguments :

  • inputImage : L'image d'entrée en couleur que vous souhaitez convertir.
  • outputImage : L'image résultante en niveaux de gris.

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat grayImage;
                convertToGray(inputImage, grayImage);

resizeImage

Description : Cette fonction redimensionne une image à une largeur et une hauteur spécifiques.

Utilisation : resizeImage(inputImage, outputImage, width, height);

Arguments :

  • inputImage : L'image d'entrée que vous souhaitez redimensionner.
  • outputImage : L'image résultante redimensionnée.
  • width : Largeur de l'image de sortie.
  • height : Hauteur de l'image de sortie.

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat resizedImage;
                resizeImage(inputImage, resizedImage, 640, 480);

applyGaussianBlur

Description : Cette fonction applique un flou gaussien à une image.

Utilisation : applyGaussianBlur(inputImage, outputImage, kernelSize);

Arguments :

  • inputImage : L'image d'entrée sur laquelle vous souhaitez appliquer le flou gaussien.
  • outputImage : L'image résultante avec le flou gaussien.
  • kernelSize : La taille du noyau de flou gaussien.

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat blurredImage;
                applyGaussianBlur(inputImage, blurredImage, 5);

detectAndDrawContours

Description : Cette fonction détecte les contours dans une image en niveaux de gris et les dessine sur une image en couleur.

Utilisation : detectAndDrawContours(inputImage, outputImage);

Arguments :

  • inputImage : L'image d'entrée que vous souhaitez détecter les contours.
  • outputImage : L'image de sortie avec les contours dessinés.

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat outputImage;
                detectAndDrawContours(inputImage, outputImage);

NoirEtBlanc

Description : Cette fonction convertit une image en couleur en une image en noir et blanc avec une seuil de binarisation.

Utilisation : NoirEtBlanc(inputImage, outputImage);

Arguments :

  • inputImage : L'image d'entrée en couleur que vous souhaitez convertir en noir et blanc.
  • outputImage : L'image résultante en noir et blanc.

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat bwImage;
                NoirEtBlanc(inputImage, bwImage);

FloydSteinbergDithering

Description : Cette fonction applique la méthode de dithering de Floyd-Steinberg pour réduire une image en niveaux de gris à une image binaire (noir et blanc) tout en conservant les détails.

Utilisation : FloydSteinbergDithering(inputImage, outputImage);

Arguments :

  • inputImage : L'image d'entrée en niveaux de gris à dither.
  • outputImage : L'image résultante en noir et blanc dithered.

Exemple : 

                Mat inputImage = imread("grayscale_image.jpg", IMREAD_GRAYSCALE);
                Mat ditheredImage;
                FloydSteinbergDithering(inputImage, ditheredImage);

Negatif

Description : Cette fonction génère le négatif d'une image, inversant les couleurs.

Utilisation : Negatif(inputImage, outputImage);

Arguments :

  • inputImage : L'image d'entrée dont vous souhaitez générer le négatif.
  • outputImage : L'image résultante en négatif.

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat negativeImage;
                Negatif(inputImage, negativeImage);

applyCustomFilter

Description : Cette fonction vous permet d'implémenter un filtre personnalisé sur une image. Dans l'exemple donné, un filtre de flou gaussien est appliqué.

Utilisation : applyCustomFilter(inputImage, outputImage);

Arguments :

  • inputImage : L'image d'entrée sur laquelle vous souhaitez appliquer le filtre personnalisé.
  • outputImage : L'image résultante après application du filtre.

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat filteredImage;
                applyCustomFilter(inputImage, filteredImage);

applyMedianBlur

Description : Cette fonction applique un filtre de flou médian à une image, utile pour réduire le bruit.

Utilisation : applyMedianBlur(inputImage, outputImage, kernelSize);

Arguments :

  • inputImage : L'image d'entrée sur laquelle vous souhaitez appliquer le filtre de flou médian.
  • outputImage : L'image résultante après application du filtre.
  • kernelSize : La taille du noyau du filtre médian (doit être impair).

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat blurredImage;
                applyMedianBlur(inputImage, blurredImage, 5);

applySobel

Description : Cette fonction applique le filtre de Sobel pour détecter les contours d'une image.

Utilisation : applySobel(inputImage, outputImage);

Arguments :

  • inputImage : L'image d'entrée sur laquelle vous souhaitez appliquer le filtre de Sobel.
  • outputImage : L'image résultante avec les contours détectés.

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat edgesImage;
                applySobel(inputImage, edgesImage);

applyCanny

Description : Cette fonction applique l'algorithme de détection de contours Canny à une image en niveaux de gris.

Utilisation : applyCanny(inputImage, outputImage, threshold1, threshold2);

Arguments :

  • inputImage : L'image d'entrée en niveaux de gris sur laquelle vous souhaitez appliquer Canny.
  • outputImage : L'image résultante avec les contours détectés.
  • threshold1 : Les seuils pour la détection de contours. Les valeurs dépendent de votre image et du niveau de détail souhaité.
  • threshold2 : Les seuils pour la détection de contours. Les valeurs dépendent de votre image et du niveau de détail souhaité.

Exemple : 

                Mat inputImage = imread("grayscale_image.jpg", IMREAD_GRAYSCALE);
                Mat edgesImage;
                applyCanny(inputImage, edgesImage, 100, 200);

rotateImage

Description : Cette fonction effectue une rotation sur une image.

Utilisation : rotateImage(inputImage, outputImage, angle);

Arguments :

  • inputImage : L'image que vous souhaitez faire pivoter.
  • outputImage : L'image résultante après la rotation.
  • angle : L'angle de rotation en degrés.

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat rotatedImage;
                rotateImage(inputImage, rotatedImage, 45.0);

adjustBrightness

Description : Cette fonction ajuste la luminosité d'une image.

Utilisation : adjustBrightness(inputImage, outputImage, brightness);

Arguments :

  • inputImage : L'image sur laquelle vous souhaitez ajuster la luminosité.
  • outputImage : L'image résultante avec la luminosité ajustée.
  • angle : La valeur d'ajustement de la luminosité (positive ou négative).

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat brightenedImage;
                adjustBrightness(inputImage, brightenedImage, 30);

adjustContrast

Description : Cette fonction ajuste le contraste d'une image.

Utilisation : adjustContrast(inputImage, outputImage, contrast);

Arguments :

  • inputImage : L'image sur laquelle vous souhaitez ajuster le contraste.
  • outputImage : L'image résultante avec le contraste ajusté.
  • angle : La valeur d'ajustement du contraste (positive ou négative).

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat highContrastImage;
                adjustContrast(inputImage, highContrastImage, 2.0);

addVignette

Description : Cette fonction ajoute un effet de vignette à une image, assombrissant les coins.

Utilisation : addVignette(inputImage, outputImage, factor);

Arguments :

  • inputImage : L'image à laquelle vous souhaitez ajouter une vignette.
  • outputImage : L'image résultante avec l'effet de vignette.
  • factor : Le facteur de l'effet de vignette (0.0 pour aucun, 1.0 pour maximal).

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat vignettedImage;
                addVignette(inputImage, vignettedImage, 0.5);

compressImage

Description : Cette fonction compresse une image en utilisant le format JPEG.

Utilisation : compressImage(inputImage, compressedImage, quality);

Arguments :

  • inputImage : L'image que vous souhaitez compresser.
  • compressedImage : Un vecteur où l'image compressée sera stockée.
  • quality : La qualité de compression de l'image (de 0 à 100, 100 étant la meilleure qualité).

Exemple : 

                Mat inputImage = imread("image.jpg");
                std::vector compressedData;
                compressImage(inputImage, compressedData, 90);

deompressImage

Description : Cette fonction décompresse une image au format JPEG.

Utilisation : decompressImage(compressedImage, outputImage);

Arguments :

  • compressedImage : Le vecteur contenant les données de l'image compressée au format JPEG.
  • outputImage : L'image résultante décompressée.

Exemple : 

                std::vector compressedData = ...;  // Chargez les données depuis un fichier ou une source
                Mat decompressedImage;
                decompressImage(compressedData, decompressedImage);

cropToSquare

Description : Cette fonction recadre une image en un carré en conservant la région centrale de l'image.

Utilisation : cropToSquare(inputImage, outputImage);

Arguments :

  • inputImage : L'image que vous souhaitez recadrer en carré.
  • outputImage : L'image résultante recadrée en carré.

Exemple : 

                Mat inputImage = imread("image.jpg");
                Mat squaredImage;
                cropToSquare(inputImage, squaredImage);

equalizeImageHistogram

Description : Cette fonction égalise l'histogramme d'une image en niveaux de gris, améliorant le contraste.

Utilisation : equalizeImageHistogram(inputImage, outputImage);

Arguments :

  • inputImage : L'image en niveaux de gris que vous souhaitez égaliser.
  • outputImage : L'image résultante avec l'histogramme égalisé.

Exemple : 

                Mat inputImage = imread("grayscale_image.jpg", IMREAD_GRAYSCALE);
                Mat equalizedImage;
                equalizeImageHistogram(inputImage, equalizedImage);

calculateHistogram

Description : Cette fonction calcule l'histogramme d'une image en niveaux de gris.

Utilisation : calculateHistogram(inputImage, histogram);

Arguments :

  • inputImage : L'image pour laquelle vous souhaitez calculer l'histogramme.
  • histogram : L'histogramme résultant, stocké dans une matrice.

Exemple : 

                Mat inputImage = imread("grayscale_image.jpg", IMREAD_GRAYSCALE);
                Mat histogram;
                calculateHistogram(inputImage, histogram);

displayHistogram

Description : Cette fonction affiche un histogramme sous forme d'image.

Utilisation : displayHistogram(histogram);

Arguments :

  • histogram : L'histogramme que vous souhaitez afficher sous forme d'image.

Exemple : 

                Mat histogram = ...;  // Remplacez par votre propre histogramme
                displayHistogram(histogram);
